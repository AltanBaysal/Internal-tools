# colab-tutorial

Google Colab'ı öğrenmek için hazırlanmış basit bir Jupyter notebook.

## File

- `learn_colab.ipynb` — GPU kontrolü, iki sayıyı toplama ve dosya indirme örneği

## Usage

1. [Google Colab](https://colab.research.google.com/) açın
2. **File → Upload notebook** ile `learn_colab.ipynb` dosyasını yükleyin
3. Hücreleri sırayla çalıştırın (Shift + Enter)
